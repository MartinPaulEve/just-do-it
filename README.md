# Just Do It (JDI)

A personal task management application built with Django and HTMX.

## Features

- Customisable task list categories (Work, Medical, Home, etc.) displayed as a dashboard
- Drag-and-drop panel and task reordering
- Four views: Dashboard, Today, Calendar, and All Tasks
- Sub-tasks, follow-up scheduling, file attachments, and links
- Cross-device sync via HTMX polling
- Responsive design for desktop and mobile

## Tech Stack

- Django 5.2
- PostgreSQL 17
- HTMX 2.x
- SortableJS
- FullCalendar 6.x
- Docker / Traefik

## Status

This project is a work in progress.
